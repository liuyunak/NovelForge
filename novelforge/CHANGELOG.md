# Changelog

All notable changes to NovelForge will be documented in this file.

## [1.0.0] — 2026-06-19

### Added
- Multi-Provider AI architecture (DeepSeek, OpenAI, Ollama, llama.cpp, LM Studio)
- DAG parallel Agent orchestration pipeline (Planner, Writer, DeepAudit, Polisher, Analyst)
- Full-memory system with 1M context support
- Style fingerprint extraction and style transfer
- Dual-layer audit system (FastAudit rules + DeepAudit LLM)
- One-click cover generation
- Script exporting (novel → short drama format)
- Web-based setup wizard for first-run configuration
- JWT-based authentication with PBKDF2-SHA512 password hashing
- Single-tenant and multi-tenant deployment modes
- Docker and Docker Compose support
- Windows-friendly one-click installer (install.bat)
- Comprehensive user manual (Word format, 10 chapters)
